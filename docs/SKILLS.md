# Skill Index

## grok-cli

Use the local `grok` CLI as an auxiliary agent for:

- quick second opinions
- design critique
- headless prompt-file runs
- session export and inspection
- model, MCP, plugin, and worktree diagnostics

Dependencies: `grok` CLI and a working Grok login.

## imagegen

Generate or edit images through the OpenAI Image API with a bundled Python CLI.

Dependencies: Python, `openai`, `pillow`, network access, and `OPENAI_API_KEY` for live calls.
