# Skill Index

## grok-cli

Use the local `grok` CLI as an auxiliary agent for:

- quick second opinions
- design critique
- headless prompt-file runs
- session export and inspection
- model, MCP, plugin, and worktree diagnostics

Dependencies: `grok` CLI and a working Grok login.

## agy-cli

Use the local Antigravity `agy` CLI as an auxiliary agent for:

- quick second opinions
- design critique
- disposable project builds
- headless prompt-file runs
- conversation resume and inspection
- model, plugin, sandbox, and Antigravity workflow diagnostics

Dependencies: `agy` CLI and a working Antigravity login.

## codex-cli

Use the local `codex` CLI as an auxiliary Codex agent for:

- full-permission headless implementation
- additional same-stack code review passes
- JSONL event streaming and final-message capture
- MCP, plugin, feature, and doctor diagnostics
- session resume and Codex-specific workflows

Dependencies: `codex` CLI and a working Codex login.

## imagegen

Generate or edit images through the OpenAI Image API with a bundled Python CLI.

Dependencies: Python, `openai`, `pillow`, network access, and `OPENAI_API_KEY` for live calls.
